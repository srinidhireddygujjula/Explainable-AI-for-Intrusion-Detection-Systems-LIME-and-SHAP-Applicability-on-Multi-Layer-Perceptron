🚀 Explainable AI for Intrusion Detection System (IDS)

This project focuses on building an intelligent Intrusion Detection System using a Multi-Layer Perceptron (MLP) model and enhancing its transparency with Explainable AI (XAI) techniques like LIME and SHAP.

📌 Overview

Intrusion Detection Systems play a crucial role in identifying malicious activities in network traffic. However, deep learning models often act as "black boxes."
This project addresses that issue by integrating interpretability methods to explain model predictions.

🎯 Objectives
Detect network intrusions using MLP
Improve model interpretability
Apply LIME and SHAP for explanation
Identify key features influencing predictions
Enhance trust and reliability in IDS
